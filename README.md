### 分享我从tonymacx86白嫖来的EFI

#### 首先说下我的主要配置:

* 主板: 华硕 `Prime Z370-A II`

* CPU:  Intel  `i5 9600k`

* 内存: 威刚万紫千红 `DDR4 2666 8G` X2

* 硬盘: 西部数据 `SN550 250G M.2`

* 显卡:  AMD `RX580 2304sp 4G显存` 

* 显示器:  AOC `U279PQU 27寸 4K`

* CPU散热器: 酷冷至尊 `T400i`

* 电源:  鑫谷 `GP750 额定650W`

* WI-FI/蓝牙: `BCM943602CD`
